## 一、不定积分基本公式

1. $$\int k dx = kx + C \quad (k \text{ 为常数})$$

2. 
$$
\begin{aligned}
(1) &\quad \int x^a dx = \frac{1}{a+1} x^{a+1} + C \quad (a \neq -1) \\\\
(2) &\quad \int \frac{dx}{x} = \ln |x| + C \quad (x \neq 0)
\end{aligned}
$$

3. 
$$
\begin{aligned}
(1) &\quad \int a^x dx = \frac{a^x}{\ln a} + C \quad (a > 0, a \neq 1) \\\\
(2) &\quad \int e^x dx = e^x + C
\end{aligned}
$$

4. 
$$
\begin{aligned}
(1) &\quad \int \sin x dx = -\cos x + C \\\\
(2) &\quad \int \cos x dx = \sin x + C \\\\
(3) &\quad \int \tan x dx = -\ln |\cos x| + C \\\\
(4) &\quad \int \cot x dx = \ln |\sin x| + C \\\\
(5) &\quad \int \sec x dx = \ln |\sec x + \tan x| + C \\\\
(6) &\quad \int \csc x dx = \ln |\csc x - \cot x| + C \\\\
(7) &\quad \int \sec^2 x dx = \tan x + C \\\\
(8) &\quad \int \csc^2 x dx = -\cot x + C \\\\
(9) &\quad \int \sec x \tan x dx = \sec x + C \\\\
(10) &\quad \int \csc x \cot x dx = -\csc x + C
\end{aligned}
$$

5. 
$$
\begin{aligned}
(1) &\quad \int \frac{dx}{\sqrt{1-x^2}} = \arcsin x + C \\\\
(2) &\quad \int \frac{dx}{\sqrt{a^2-x^2}} = \arcsin \frac{x}{a} + C \quad (a > 0) \\\\
(3) &\quad \int \frac{dx}{1+x^2} = \arctan x + C \\\\
(4) &\quad \int \frac{dx}{a^2+x^2} = \frac{1}{a} \arctan \frac{x}{a} + C \quad (a \neq 0) \\\\
(5) &\quad \int \frac{dx}{x^2-a^2} = \frac{1}{2a} \ln \left| \frac{x-a}{x+a} \right| + C \\\\
(6) &\quad \int \frac{dx}{\sqrt{x^2+a^2}} = \ln \left( x+\sqrt{x^2+a^2} \right) + C \\\\
(7) &\quad \int \frac{dx}{\sqrt{x^2-a^2}} = \ln \left| x+\sqrt{x^2-a^2} \right| + C \\\\
(8) &\quad \int \sqrt{a^2-x^2} dx = \frac{a^2}{2} \arcsin \frac{x}{a} + \frac{x}{2} \sqrt{a^2-x^2} + C
\end{aligned}
$$
